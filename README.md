# Natural Language Processing (NLP) Lab

Welcome to my NLP repository! This repository serves as a centralized workspace for my projects, university assignments, and experiments in Natural Language Processing, Computational Linguistics, and Generative AI. 

The goal of this repo is to track my implementation of core NLP pipelines, ranging from traditional text processing to modern Large Language Model (LLM) architectures.

---

## 🚀 Repository Structure

The project is organized by modules based on specific NLP tasks and foundational topics:

```text
├── regex/                  # Text extraction, pattern matching, and tokenization basics
├── preprocessing/          # Normalization, lemmatization, and stop-word removal
├── embeddings/             # Word vectors, TF-IDF, and Vector Databases (e.g., Chroma/Pinecone)
├── models/                 # Sequence models, Transformers, and PyTorch implementations
└── workflows/              # Advanced pipelines, RAG, and Agentic AI experiments
